1、mv initial_path(file_! file_2 / folder) target_path 
   -v 获取详细信息
   -i 覆盖询问
   -u 覆盖旧版本
   -n 不覆盖
   -b 旧文件备份，然后覆盖
   -S<xx> 为备份文件制定后缀
   -f 无条件覆盖
2、ls 
   -a 显示隐藏文件  -A 显示除开“.”和“..”以外的所有文件
   -l 显示详细信息，包括权限模式、所有者、组、文件大小、文件最后的修改时间  
   -t 按更改时间排序
   -R 显示递归文件
   -m 用“,”区隔每个文件
3、mkdir
   -m 设置权限
   -p 多层目录创建

   rmdir(删除空目录)
   -p 删除多层目录
4、cd 
    /~ 主目录
   -   之前的目录
   !$ 上个命令的参数作为cd参数使用
5、df 显示磁盘空间
   -h 以可读性较高的方式显示
   -k -m -g 单位
6、u：User g:Group o:Other a:All
   r：读取，4 
   w：写入，2 
   x：执行，1

   chmod 变更文件/目录权限 u+（-=）rwx/700
   -R 递归处理
   -v 显示过程
7、vim
   行首：Shift+6("^") | 0  行末：n + Shift + 4("$")   
   上下左右（行列）：kjhl nk(向上移动n行)
   前后（单词）：wb nw(向前移动n个单词,到单词第一个字母)
                 e/ge ne(...,到单词最后一个字母)
   匹配(),{},[]等：%
   开头：gg  结尾：shift + g
   dd 删除一行 
   u  撤销上一次的操作
   /+字符  搜索词  n到下一个字符
   e! 放弃修改文件内容，重新载入该文件编辑
8、ssh
   实现远程登录服务器
   ssh IP地址（默认使用当前用户登录）
   -l:指定用户名/user@IP地址
   -p:指定端口登录
   -c：接收或者发送的数据会被压缩
   -v:调试模式
   -b:指定本机IP进行连接
     ：在地址后直接加命令即可执行
9、cp
   -f 强行复制，不论是否存在
   -i 覆盖前询问用户
   -p 保留源文件的属性
   -R/r 递归处理，将目录下所有文件和子目录一并处理
   -b  覆盖已存在的文件前将目标文件备份   
10、dirs 显示当前目录栈所有记录    -l 以完整格式显示目录   -p 一个目录一行
    pwd  以绝对路径显示当前工作目录  
11、source  用于执行文件，将文件的内容当做shell 命令逐行执行
    source filename    该命令可用“.”代替
    source与sh及./filename的区别在于，sh或./filename会创建一个子shell，在子shell中执行脚本语句，不会带回父shell。source则是直接在当前shell里执行。  
12、rm 删除文件，不可恢复
    -f 强制删除
    -i 询问用户
    -r/-R 递归删除
    -v 显示详细信息
    --preserve--root  不对根目录进行递归操作
13、find 查找文件的命令
    -name  按照名字
    -iname 忽略大小写
    -amin<分钟> 指定时间内被存取过的文件
    -empty 寻找大小为0 byte的文件
    -path<范本样式> 指定字符串作为寸照目标的范本样式
    -iregex  同上，忽略大小写
    ！ 否定参数
    -type ?
14、touch a、创造一个新的文件  b、将已有文件时间标签更改为当前时间
    -a 只改存取时间
    -d 使用指定的时间，而非限制的时间
15、cat 显示文件内容
    -n 对所有输出的行数编号
    -b 对行数编号，空白行不编号        
16、grep 全面搜索正则表达式并把行打印出来
    -A<显示列数> 显示符合条件的那一行之外，显示该行之后的内容
    -b  与-A相反，显示该行之前的内容
    -c  计算符合范本样式的行数
    -H  在显示符合范本样式的那一列之前，标示该列的文件名称。
    -i  忽略大小写的差别
    -n  显示行号
    -R/-r 查找的是目录
    -E  将范本样式为延伸的普通表示法来使用，意味着使用能使用扩展正则表达式。???
    -o  只输出文件中匹配到的部分    
17、ps 用于报告当前系统的进程状态。搭配kill命令随时中断、删除不必要的程序
    -A 显示全部程序
    -a 显示目前执行的程序
    c 列出真正的指令名称，而不包含路径
