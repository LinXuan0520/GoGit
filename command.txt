1、mv initial_path(file_! file_2 / folder) target_path 
   -v 获取详细信息
   -i 覆盖询问
   -u 覆盖旧版本
   -n 不覆盖
   -b 旧文件备份，然后覆盖
   -S<xx> 为备份文件制定后缀
   -f 无条件覆盖
2、ls 
   -a 显示隐藏文件  -A 显示除开“.”和“..”以外的所有文件
   -l 显示详细信息，包括权限模式、所有者、组、文件大小、文件最后的修改时间  
   -t 按更改时间排序
   -R 显示递归文件
   -m 用“,”区隔每个文件
3、mkdir
   -m 设置权限
   -p 多层目录创建

   rmdir(删除空目录)
   -p 删除多层目录
4、cd 
    /~ 主目录
   -   之前的目录
   !$ 上个命令的参数作为cd参数使用
5、df 显示磁盘空间
   -h 以可读性较高的方式显示
   -k -m -g 单位
6、u：User g:Group o:Other a:All
   r：读取，4 
   w：写入，2 
   x：执行，1

   chmod 变更文件/目录权限 u+（-=）rwx/700
   -R 递归处理
   -v 显示过程
7、vim
   行首：Shift+6("^") | 0  行末：n + Shift + 4("$")   
   上下左右（行列）：kjhl nk(向上移动n行)
   前后（单词）：wb nw(向前移动n个单词,到单词第一个字母)
                 e/ge ne(...,到单词最后一个字母)
   匹配(),{},[]等：%
   开头：gg  结尾：shift + g 
8、ssh
   实现远程登录服务器
   ssh IP地址（默认使用当前用户登录）
   -l:制定用户名/user@IP地址
   -p:制定端口登录
   -c：接收或者发送的数据会被压缩
   -v:调试模式
   -b:制定本机IP进行连接
     ：在地址后直接加命令即可执行
9、cp
   -f 强行复制，不论是否存在
   -i 覆盖前询问用户
   -p 保留源文件的属性
   -R/r 递归处理，将目录下所有文件和子目录一并处理
   -b  覆盖已存在的文件前将目标文件备份
